# Selenium Bot with VPN Rotation

Bot Selenium otomatis dengan fitur VPN rotation, user agent rotation, dan monitoring web.

## Fitur

- ✅ Rotasi User Agent setiap sesi
- ✅ VPN Extension auto-install (Touch VPN, Hotspot Shield)
- ✅ Google location rotation  
- ✅ Data leak checking
- ✅ Smart scrolling dengan durasi random
- ✅ Auto close ads dengan multiple selectors
- ✅ Cache clearing otomatis
- ✅ Web monitoring dashboard
- ✅ Deploy ready untuk Render
